About Our Smart Agriculture System

Our project, Smart Agriculture for Organic Farming, combines the power of AI, IoT, and real-time data analysis to monitor farm conditions and help farmers make informed decisions. Designed for simplicity and effectiveness, this system improves crop quality while promoting sustainable practices.

Sensor Integration

We use a range of calibrated sensors to measure critical parameters in real-time:

  • Soil Moisture: Helps prevent under/overwatering.
  • Light Intensity: Assesses the available sunlight.
  • Soil pH: Determines soil acidity for optimal crop growth.
  • Soil Temperature: Monitors root-zone heat for better seed germination.
  • Ambient Temperature & Humidity: Tracks environmental conditions for overall plant health.

Our Technology Stack

The project is built using:

  • Microcontroller: Raspberry Pi Pico W with MicroPython
  • Web Backend: Python Flask + SQLite3
  • Frontend: HTML, CSS, JavaScript, Tailwind CSS
  • Communication: Wi-Fi data push to the Flask server
  • AI Recommendation (Upcoming): NLP-based insights for organic practices

Future Goals

We plan to extend the system with:

  • AI-based crop and fertilizer recommendations
  • Weather integration for predictive alerts
  • Mobile app for real-time monitoring
